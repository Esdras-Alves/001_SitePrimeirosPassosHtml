História da Web

(Voltar)

Primeiros Computadores

Na 1ª geração de computadores temos o Havard Mark I (1944), Colossus (1946) e ENIAC (1946). Todos imensos, e usavam válvulas.

Na 2ª geração (1959-1965), houve uma grande evolução com a substuição das válvulas por Transistores, diminuido muito o tamanho do computador.

Na 3ª geração (1965-1970), foi criado o Circuito Integrado, em substuição ao Transistor, diminuindo ainda mais o tamanho.

A 4ª geração, marcada a partir de 1971, teve como característica inicial a criação do mouse, pela Apple, e se estende até os dias atuais, com os Smartphones e computadores com multiprocessadores.

Internet - Como Surgiu

Apesar dos conceitos militares por trás da internet surgirem por volta dos anos 50, foi na década de 60, no contexto da polarização mundial causada pela Guerra Fria, entre os Estados Unidos e URSS, que a internet começou a se desenvolver. Os EUA criaram o sistema de descentralização de suas informações no Pentágono para evitar que possíveis ataques causassem a perda irreparável de documentos do governo.

O engenheiro Joseph Licklider foi o criador da ARPANET, a rede de conexão da DARPA, Agência de Projetos de Pesquisa Avançada dos Estados Unidos. A primeira conexão estabelecida foi feita em 29/10/1969 entre a Universidade da Califórnia e o Instituto de Pesquisa de Stanford.

A palavra enviada foi "LOGIN", mas só chegou "LO".

Na década de 70 foram criados os protocolos TCP/IP por Vinton Cerf e Robert Kahn.

Em 1889, Tim Berners-Lee viu a oportunidade de unir hipertexto com TCP/IP e criar a World Wide Web.

E assim foi criado o primeiro navegador, chamado WorldWideWeb, em 06/08/1991.


O Que São Clients

(Voltar)

Conceito de Client no "linguajar" Web

Basicamente é o lado do usuário final, onde eu e você, por meio de um navegador, acessamos algo. Pode ser através de um celular, computador, videogame, etc.

Falando Sobre Navegadores

Também chamados de browsers, são programas criados por empresas, para abrir/executar arquivos. Seguem padrões da W3C. Iniciou com o MOSAIC e hoje temos uma variedade de navegadores disponíveis.

Aplicações Web

São soluções criadas que possuem a internet como meio de comunicação entre Client x Server, não sendo necessário a instalação na máquina.


O Que São Servers

(Voltar)

Sobre Servidores

Computador com alta capacidade de processamento que conecta outros computadores através de uma rede, como uma rede local, ou a web de forma geral. Pode ser de arquivos, segurança (Firewall), streaming, e-mail, web. O servidor pode ser pequeno ou até mesmo ocupar grandes salas.

Sobre Hardwares e Softwares em Servidores

Softwares utilizados: Sistemas Operacionais (como Windows Server, Ubuntu, CentOS, Fedora, etc), Monitoramento e Servidores Web (como Apache e NGINX).

Hardwares utilizados: Armazenamento (discos rígidos, SSDs), Memória, Processadores.

O cliente faz a requisição HTTP (protocolo de transferência de hipertexto), que chega ao servidor, e lá o software do servidor devolve em formato HTML, CSS, Javascript, Imagens, Texto,etc.

Hospedagem de Sites

Quando contratamos o serviço de hospedagem de sites, estamos contratando um servidor web para disponilizar nosso site ali. Dessa forma, qualquer um que queira, consigue acessar.


Linguagens de Programação

(Voltar)

O que é Linguagem de Programação

Linguagem de Programação é uma linguagem escrita e formal que especifica um conjunto de instruções e regras usadas para gerar programas (software).

5 linguagens mais usadas, segundo RedMonk:

  1. Javascript
  2. Python
  3. Java
  4. PHP
  5. C#

Linguagem Server-Side

Linguagens executadas no servidor, como PHP, Ruby, C# e Java, e Javascrip.

Linguagem Client-Side

São linguagens executadas no lado Client, como Javascript, HTML e CSS.

Como assim? Javascrip é executado tanto no lado Client quando no lado Server?

Isso mesmo, o Javascript possui uma ramificação que se baseia no interpretador v8 do Google, que pode ser executada no lado servidor, chamada Node.js.

Estrutura Básica de um HTML

  • O documento deve iniciar com a tag "html", seguida de "head" com o "title" (título da página) e "body", onde ficará o conteúdo da página.
  • Títulos (h1), subtítulos (h2 em diante) e parágrafos (p).
  • Negrito (strong ou b), itálico (i), sublinhado (u), marcado (mark), elevar algum caractere* (sup, e o caractere).
  • Listas ordenadas com "ol" e não ordenadas com "ul". Cada item da lista recebe a tag "li".
  • A tag "a href="">" serve para colocar links externos (com endereços Web) ou internos (através do id) na página. Os atributo "title" serve para definir um texto a ser exibido ao passar com o mouse no link, e o atributo "target" serve para definir se o link externo será aberto na mesma aba, ou em outra (com o valor blank).

Para conhecer mais tags e seus usos acesse o site: w3schools.com/TAGS/
